# 날씨앱

## 최소 구현조건

- Open API를 이용해서 날씨 정보를 가져오고 위치 그리고 날씨 정보(현재 기온, 당일의 최저 기온, 당일의 최고 기온, 시간대 별 기온)를 화면에 보여준다.
- 앱 첫 진입 시 사용자의 현재 위치를 감지하여 그 위치의 날씨 정보를 보여준다.
- 유저가 원하는 장소를 검색(기초 자치단체, 구, 동에 상관없이 검색 가능)하고 그 장소의 날씨 정보를 조회한다.
  ex) “서울특별시”, “종로구”, “청운동” 등으로 시, 군, 구, 동 모든 단위로 검색 가능하며, 검색어 입력 시 매칭되는 장소 리스트를 표시하고 사용자가 리스트에서 장소를 선택할 수 있다. 해당 장소의 날씨 정보가 없는 경우 “해당 장소의 정보가 제공되지 않습니다.”를 UI에 명시한다.
- 검색한 장소를 즐겨찾기에 추가하거나 삭제할 수 있다.
- 즐겨찾기에는 최대 6개의 장소를 추가할 수 있고 추가한 장소는 카드 UI 형태로 보여준다.
- 즐겨찾기에 추가된 장소의 이름(별칭)을 수정한다.
- 즐겨찾기에 추가된 장소 카드에는 현재 날씨 정보, 당일의 기온(최저, 최고)을 보여준다.
- 즐겨찾기 카드를 클릭하면 해당 장소의 상세 페이지로 이동한다.

### 장소 데이터

[korea_districts.json](./korea_districts.json)
